# pokedex-frontend

Pokedex frontend scaffold with React 19, Vite 6, TypeScript 5, Tailwind v4, shadcn/ui, and React Router v7.

## Stack
- React 19, TypeScript, Vite, shadcn

## Conventions
- Functional components with hooks
- Path alias: @/* → src/*
- Components in src/components/
- Hooks in src/hooks/
- Types in src/types/
